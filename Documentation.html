Zen Programming Language

A minimal, C-inspired procedural language with concise syntax

Introduction

Zen is a procedural language based on C with simplified and shortened commands. It uses functional and conditional (statement) expressions.

Command Types

  • Function: Executes an action (e.g., printw "Hello";)
  • Statement: Evaluates a condition (e.g., streq "a", "b";)

Core Syntax

deflib "str";
defstr "a";
str "a", "hello";
printr "a";

Formatting

Use ^ to insert variable references inside other commands.

str "a", "z";
str "b", "^a";
add "d", "c";
